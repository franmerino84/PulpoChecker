# PulpoChecker
App to notify when pulpo.es has groups of a platform below a price

It accepts the parameter -p followed by a list of pairs platform-maxValue.

Example: -p spotify-3.5;hbo-4.2;filmin-3.8

This will notify in windows when there's an available group with a cost equal or below 3.5€ for Spotify, equal or below 4.2€ for HBO and equal or below 3.8€ for Filmin.
